 Program Description
This Python program generates an identity card directly in the console.  
The user enters several personal details such as full name, place and date of birth, nationality, height, weight, gender, address, and profession.  
The program automatically calculates and generates a simple identification number.  
All the entered information is then displayed as a complete identity card in a clear and readable format.  
This project demonstrates how to handle user input, perform basic calculations, and display results immediately in the console.
